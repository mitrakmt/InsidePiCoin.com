# Pi Network Announces Mainnet Launch Date

In exciting news for cryptocurrency enthusiasts, Pi Network has officially set the launch date for its long-awaited Open Mainnet. The transition to the Open Network is scheduled for Q1 2025, marking a significant milestone for the project which has been in development since 2019.

This launch will allow Pi Coin to be fully tradeable on exchanges, potentially opening up new avenues for the millions of Pi Network users globally. The community is abuzz with anticipation, with many speculating on the future value and utility of Pi Coin in the broader crypto ecosystem.

Pi Network has already surpassed key milestones, with over 18 million KYC'd users and a vibrant ecosystem of applications ready to integrate with the mainnet. This development comes after several extensions and a focus on ensuring the network's security and user verification processes.
